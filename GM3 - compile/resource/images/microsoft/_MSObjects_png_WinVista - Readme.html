Visual Studio Image Library: Objects (.png Format, Windows Vista style)

32-bit Art
This is 32-bit art containing an alpha channel for transparency.

These images are offered in various sizes although only one size is displayed in the table below. Refer to the "Sizes" column to see which sizes are available; the same usage restrictions apply to all sizes of a given image.

Use Restrictions
As part of a visual language, the following images (or any part of the images) must be used in a manner consistent with the name of the image file, or as described in the usage notes below:
